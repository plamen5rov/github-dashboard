# Feature Ideas

> Future enhancements for GitHub Dashboard — captured for later consideration.

## Inline README Preview
Click a repo card to expand an inline README preview directly in the feed. README text is already fetched via GraphQL, just needs a UI to show it.

## Keyboard Shortcuts
- `b` — toggle bookmark on focused repo
- `s` — focus search input
- `?` — show help overlay
- `j` / `k` — navigate cards
- Low effort, big UX win.

## Export Collections
Copy bookmarks/collections as Markdown list or JSON to share with others.

## Similar Repos
"Users who liked this also liked..." — based on shared topics/language from your own bookmarks. Pure localStorage, no backend.

## Surprise Me Button
Random repo button — fetch a random page of search results and pick one.

## PWA / Offline Support
Use `vite-plugin-pwa` to cache the app shell. Works without internet after first load.
